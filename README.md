# Lógica de Programação - TADS 🚀

Repositório dedicado aos estudos de lógica de programação e algoritmos, focado na transição do VisuAlg para Python.

## 📂 Estrutura Sequencial (Concluída)
Nesta seção, pratiquei a entrada de dados, processamento matemático e saída formatada.

| Exercício | Descrição | Conceitos Chave |
| :--- | :--- | :--- |
| **Soma** | `exercicio_soma.alg` | Operadores aritméticos básicos. |
| **Produtos** | `exercicios_produtos.alg` | Manipulação de múltiplas variáveis. |
| **Terreno** | `exercicio_terreno.alg` | Cálculo de área e preço por m². |
| **Círculo** | `exercicio_circulo.alg` | Uso da constante `PI` e exponenciação. |
| **Retângulo** | `exercicio_retangulo.alg` | Geometria, raiz quadrada e diagonais. |
| **Medidas** | `exercicios_medidas.alg` | Áreas de quadrado, triângulo e trapézio. |
| **Troco** | `exercicio_troco.alg` | Precedência de operadores e lógica comercial. |
| **Pagamento** | `exercicio_pagamento.alg` | Interação entre strings (nomes) e reais. |
| **Idades** | `exercicio_idades.alg` | Entrada de múltiplos dados e concatenação. |
| **Consumo** | `exercicio_consumo.alg` | Divisão simples e rendimento de combustível. |
| **Duração** | `exercicio_tempo.alg` | Divisão inteira (`\`) e resto da divisão (`%`). |

## 🛠️ Tecnologias e Ferramentas
* **Ambiente:** Ubuntu Linux (WSL2)
* **Linguagens:** VisuAlg (.alg) e Python (.py)
* **Controle de Versão:** Git & GitHub
